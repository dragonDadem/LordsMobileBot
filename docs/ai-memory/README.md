# AI Memory System

This directory contains the shared memory and coordination system for multiple AI agents working on LordsMobileBot.

## Structure

- **core/**: Live project state and critical files
- **history/**: Archived session reports
- **knowledge/**: Permanent lessons and patterns
- **templates/**: Reusable templates

## Enforcement
All agents must follow the rules in `memory_guidelines.md` and respect `LOCK.json`.

**Current Objective** is defined in `core/current_objective.md`
