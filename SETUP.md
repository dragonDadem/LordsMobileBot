# دليل إعداد بوت لوردس موبايل (Lords Mobile Bot)

هذا المشروع مصمم ليكون أداة احترافية للأتمتة. يرجى اتباع الخطوات التالية بعناية لضمان عمل البوت بشكل صحيح.

## 1. إعداد المحاكي (LDPlayer)
يجب ضبط المحاكي بدقة ليعمل الكشف البصري بشكل صحيح:
- **الدقة:** يجب أن تكون `1600x900`.
- **ADB:** تأكد من تفعيل تصحيح أخطاء USB (USB Debugging) في إعدادات المطور داخل المحاكي.

## 2. إعداد الصور (القوالب)
البوت لا يتعرف على الموارد تلقائياً بدون صور مرجعية.
- اذهب إلى المجلد `assets/templates/`.
- قم بقص صور صغيرة جداً وواضحة لأيقونات الموارد من الخريطة.
- **تسمية الملفات:** استخدم الصيغة `ResourceName_lvX.png`.
  - مثال: `Gold_lv5.png`, `Wood_lv3.png`, `Ore_lv5.png`.

## 3. إحداثيات الأزرار
بما أن أماكن الأزرار قد تختلف قليلاً حسب واجهة اللعبة لديك:
- افتح ملف `dispatch/manager.py`.
- ابحث عن قاموس `self.coords`.
- قم بتحديث قيم (X, Y) لكل زر (Gather, March, Confirm) بناءً على مكانها في شاشتك.

## 4. التشغيل
1. افتح نافذة الأوامر (Terminal/CMD) في مجلد المشروع.
2. ثبت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
3. شغل البوت:
   ```bash
   python main.py
   ```

## 5. واجهة التحكم
- **START BOT:** لبدء عملية المسح والجمع.
- **PAUSE:** لإيقاف البوت مؤقتاً (سيحتفظ بالحالة).
- **STOP BOT:** لإيقاف البوت تماماً.
- **EVENT LOG:** يعرض لك كل ما يفعله البوت في الوقت الفعلي.

---
**ملاحظة أمنية:** البوت يحتوي على ميزات "Humanization" لتقليد سلوك البشر، ولكن دائماً استخدم الأتمتة بحذر.
